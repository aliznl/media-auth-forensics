# Contributing

## Principles
- Reproducible improvements only
- Any detection change must include tests and evaluation notes

## Setup
- Python 3.9+
- `pip install -r requirements.txt`
- `pip install -e .`

## PR requirements
- Clear summary and rationale
- Tests updated or added
- Metrics reported when relevant
- Document known limitations

## Style
- Small, testable functions
- Docstrings and type hints where practical
- Structured logging for pipeline decisions
